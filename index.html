HTML

  • A programming language used to create the structure of a webpage
  • Made up of elements that provide the content and framework for a webpage
  • Composed of two main elements: the head and the body
  • The head contains the metadata of the webpage and is not visible to the user
  • The body contains the visible content seen by the user
  • HTML elements are usually composed of angle brackets which represent the opening and closing tags
  • Semantic HTML elements describe its content and the role it plays on a webpage
  • Child elements are elements nested inside other elements

CSS

  • A programming language used to define how the webpage content is presented to users
  • Defines: colors, aesthetics, visual layout, and organization
  • There are three ways to style a webpage using CSS: incline CSS, internal CSS style sheet, and external CSS style sheet
  • Incline CSS: styling an element directly in the HTML file
  • Internal CSS style sheet: contains CSS rules in the head section of the HTML file
  • External CSS style sheet: contains all CSS rules in a seperate file
  • Declartations contain the CSS property we want to apply and the value of the property
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
